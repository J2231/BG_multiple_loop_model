# BG_multiple_loop_model
Basal ganglia multiple-loop computational model used in [A Model of Interacting Basal Ganglia and Cortico-Thalamocortical Loops Accounts for Key Temporal Firing Patterns in a Delayed Choice Task], submitted to PLOS Computational Biology.


## Contents
BG_multiple_loop_model_PlosCompBiol.ipynb  
The notebook runs the basal ganglia network model and generates the simulated neural activity plots used in the model panels of the paper's figures.

## Requirements
Python 3
numpy
matplotlib

## Running the model
Open the notebook in Google Colab, Jupyter Notebook, or JupyterLab and run all cells to reproduce the simulations and figures.
